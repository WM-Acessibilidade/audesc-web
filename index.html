Sobre esta versão

Esta versão do Audesc foi simplificada para funcionar como catálogo web de repositórios no formato do Memorial MPF.

Os links do catálogo abrem a página principal do repositório. Já a leitura de QR code abre diretamente o link específico do item identificado.

Arquivo de catálogo

Os repositórios são lidos de data/repositories.json.

Somente categorias com pelo menos um repositório aparecem para o usuário.

Ler QR code
