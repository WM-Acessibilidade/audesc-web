Sobre esta versão

Esta versão do Audesc foi simplificada para funcionar como catálogo web de repositórios no formato do Memorial MPF.

O aplicativo lê um arquivo específico com as categorias e os links dos repositórios. Ao abrir um repositório, o usuário é levado diretamente à página principal desse repositório, que já contém seu próprio player e a lista de itens.

Arquivo de catálogo

Os repositórios são lidos de data/repositories.json.

Somente categorias com pelo menos um repositório aparecem para o usuário.
